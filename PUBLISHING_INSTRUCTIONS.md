# Publishing Instructions

1. Create a new GitHub repository, for example `JordanChevalleyFormalization`.
2. Upload all files inside `shareable_proof_repo/`.
3. Do not upload `.lake/`.
4. Do not upload any API keys, tokens, authentication files, or private credentials.
5. After publishing, copy the repository URL.
6. Add the URL to the Assignment 2 submission email as the optional shareable proof link.

## GitHub Pages

1. Upload all files inside `shareable_proof_repo/` to a GitHub repository.
2. Go to repository Settings.
3. Go to Pages.
4. Under "Build and deployment", choose "Deploy from a branch".
5. Select branch `main` and folder `/root`.
6. Save.
7. Wait for GitHub Pages to deploy.
8. The public proof webpage should be:

```text
https://<your-username>.github.io/JordanChevalleyFormalization/
```

9. Open the page and click "Open in live Lean" to check that it opens `Formalization.lean` in the live Lean editor.

Suggested email line:

```text
Optional shareable proof link:
https://github.com/<your-username>/JordanChevalleyFormalization
```
